# 경제 시스템 기획서 (Economy System Design)

## 1. 개요
단일 소프트 커런시(골드)를 중심으로 한 경제 설계. 스테이지 클리어와 데일리 시스템(출석·챌린지)이 소득원이며, 부스터와 코스메틱이 소비처입니다. 코스메틱 시스템 도입으로 100스테이지 완주 이후에도 장기적인 골드 소비 목표를 제공합니다.

---

## 2. 재화 종류 (소프트 커런시)

| 재화 | 테마 명칭 | 역할 | 획득처 |
| :---: | :---: | --- | --- |
| **기본 재화** | **골드 (Gold)** | 부스터 구매, 코스메틱 잠금 해제, 교착 상태 탈출에 사용. | 스테이지 클리어, 챕터 마일스톤, 데일리 로그인, 데일리 챌린지, 업적 보상. |

## 3. 보상 공식 (골드 소득원)

### 3.1. 스테이지 클리어 보상 (고정)

| 챕터 | 스테이지 범위 | 최초 클리어 보상 | 재클리어 보상 |
| :---: | :---: | :---: | :---: |
| Ch.1 | Lv. 1–20 | 30골드 | 없음 |
| Ch.2 | Lv. 21–40 | 35골드 | 없음 |
| Ch.3 | Lv. 41–60 | 40골드 | 없음 |
| Ch.4 | Lv. 61–80 | 45골드 | 없음 |
| Ch.5 | Lv. 81–100 | 50골드 | 없음 |

- 이동 횟수(Moves)는 스테이지별 랭킹 산정에만 사용됩니다.
- 챕터 마일스톤 보상 상세는 `signal_sort_content_design_kr.md §3` 참조.

### 3.2. 데일리 시스템 소득원 (신규)

| 소득원 | 보상 | 빈도 | 월 추산 |
| :--- | :--- | :---: | :---: |
| 데일리 로그인 Day 1–6 | 30~150골드 | 매일 | ~2,250골드 |
| 데일리 로그인 Day 7 | 300~350골드 | 주 1회 | ~1,325골드 |
| 데일리 챌린지 클리어 | 100골드 | 매일 | ~3,000골드 |
| 챌린지 스트릭 3일 | +50골드 | 주 2회 | ~400골드 |
| 챌린지 스트릭 7일 | +200골드 | 주 1회 | ~800골드 |

- **월 데일리 시스템 추산 소득 (풀 참여 기준)**: 약 **7,775골드/월**

## 4. 경제 소비처 (Consumption Sinks)

### 4.1. 인게임 부스터 (즉석 구매)
- `Undo (되돌리기)`: **무료 · 무제한**
- `Shuffle (셔플)`: **300골드**
- `Add Lane (레인 추가)`: **500골드** (Stuck 팝업 광고 시청 시 무료 1회/스테이지)

### 4.2. 코스메틱 시스템 (신규 — 주요 장기 소비처)
코스메틱 구매 가능 목록 및 가격은 `cosmetic-system-design.md` 참조.

| 코스메틱 카테고리 | 구매 가능 총 골드 |
| :--- | :---: |
| Chip Skin (골드 구매 가능분) | 6,100골드 |
| Lane Skin (골드 구매 가능분) | 7,600골드 |
| Board Skin (골드 구매 가능분) | 10,500골드 |
| **총 코스메틱 소비 가능 금액** | **24,200골드** |

### 4.3. 아바타 잠금 해제
`social-ranking-design.md §7` 및 `avatar.csv` 기준. 유료 아바타: 200~500골드/개.

## 5. 경제 시뮬레이션 요약

### 5.1. 100스테이지 완주 플레이어 (데일리 미참여 가정)

| 소득원 | 골드 |
| :--- | :---: |
| 스테이지 클리어 합계 (20×5챕터) | 4,000 |
| 챕터 마일스톤 합계 | 6,100 |
| **총 획득** | **10,100** |

| 소비처 | 골드 (추산) |
| :--- | :---: |
| Shuffle 20회 | 6,000 |
| Add Lane 골드 구매 2회 | 1,000 |
| **총 소비** | **7,000** |

- **100스테이지 완주 순 잔액**: 약 **3,100골드**
- Chip Skin 일부 구매 가능한 잔액 (셔플 단가 300골드 상향으로 부스터 싱크 증가, 코스메틱 가용 잔액 축소).

### 5.2. 1개월 데일리 참여 플레이어 (100스테이지 완주 후)

| 소득원 | 월 골드 |
| :--- | :---: |
| 데일리 시스템 (풀 참여) | 7,775 |
| 업적 (잔여 달성) | 1,000–3,000 |
| **월 소득 추산** | **~9,000** |

- 코스메틱 풀 컬렉션(24,200골드)까지 데일리 풀 참여 기준 약 **3개월** 소요.
- 이 수치는 의도된 장기 플레이 동기 유지 목표임.

## 6. 데이터 통합 (Data Integration)
- **`reward/` 도메인**: 모든 골드 보상은 `reward_group.csv`에 정의.
- **`shop/` 도메인**: 부스터 비용은 `shop_item.csv`, 코스메틱 가격은 `cosmetic_item.csv`.
- **`achievement/` 도메인**: 업적 보상은 `achievement.csv`.
- **`player_currencies` 테이블**: `user_id` 및 `currency_id`(골드 ID: 1001)별 잔액 저장.
