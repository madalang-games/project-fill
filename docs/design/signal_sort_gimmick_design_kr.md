# Signal Sort 기믹 설계서 (Signal Sort Gimmick Design)

본 문서는 `project-fill` Signal Sort의 챕터별 신규 기믹 상세 규칙, 시각 가이드, 부스터 상호작용 매트릭스를 정의합니다.

---

## 1. Locked Lane (잠긴 레인) — Chapter 2 도입 (Lv.21~)

### 규칙
- 레인 상단에 자물쇠 스킨이 씌워져 있어 잠금 상태에서는 칩을 배치할 수 없습니다.
- **잠금 해제 조건**: 스테이지마다 지정된 특정 Signal Type의 Complete Set이 Signal Panel에 등록되면 잠금이 해제되어 일반 빈 레인으로 사용할 수 있게 됩니다.
- 잠금 상태에서는 Incoming 칩 배치 불가. Complete Set 완성 시 자동 해제.

### 시각
- 잠금 상태: 레인 상단 자물쇠 아이콘 + 해제 조건 Signal Type 색상 뱃지 표시.
- 해제 연출: 자물쇠 아이콘 해제 애니메이션 + 레인 테두리 플래시.

---

## 2. Blind Lane (블라인드 레인) — Chapter 3 도입 (Lv.41~)

### 규칙
- 레인에 쌓인 칩 중 **Top Chip만 색상·타입이 공개**되고, 하단 칩들은 뒷면(회로 기판 패턴)으로 표시됩니다.
- **A-R01~A-R07 기본 규칙은 동일하게 적용**됩니다. 정보 은닉만 추가됩니다.
- **공개 타이밍**: 현재 Top Chip이 이동하여 아래 칩이 새로운 Top Chip이 되는 순간 즉시 색상·타입이 공개됩니다.

### 시각
- 가려진 칩: 반도체 기판 회로 패턴 텍스처 (게임 테마와 일치).
- 공개 연출: 뒷면 → 앞면 플립 애니메이션 (0.15초).
- 레인 마킹: 레인 테두리에 물음표(?) 아이콘 또는 점선 테두리.

---

## 3. Relay Node (릴레이 노드) — Chapter 4 도입 (Lv.61~)

### 규칙
- 상단 Signal Panel에 **완성 순서 제약**이 지정됩니다.
- 규칙: 예고된 **A 타입 Complete Set이 먼저 Signal Panel에 등록**되어야 B 타입의 세트가 흡수될 수 있습니다.
- 순서가 어긋난 Complete Set 달성 시 해당 레인은 **대기 상태(Pending)**에 진입합니다.
- 대기 상태 레인: Capacity=4, 칩 4개 충전 상태이므로 물리적으로 Incoming 칩 배치 불가.

### 대기 상태 시각
- 레인 테두리: 앰버색 점선 테두리 + 잠금 아이콘.
- 대기 순번 뱃지: 레인 상단 `WAIT #N` 뱃지.
- Signal Panel: 해당 타입 노드가 반투명 점멸(점등 대기 상태) 표시.

---

## 4. Overload Chip (과부하 칩) — Chapter 5 도입 (Lv.81~)

### 규칙
- 극성이 너무 강해 **빈 레인에는 단독으로 내려놓을 수 없고**, 오직 **같은 Signal Type의 칩이 Top Chip인 레인으로만 이동 가능**한 특수 칩입니다.
- Overload Chip이 Top Chip인 상태에서 다른 레인으로 이동할 때는 일반 규칙(A-R02) 적용.

### 시각
- 칩 테두리: 과부하 글로우 효과 (붉은 오렌지색 펄스).
- 배치 불가 레인 시도 시: 기존 A-R07 피드백(Shake + 빨간 테두리) 동일 적용.

---

## 5. 부스터 상호작용 매트릭스

| 기믹 | Shuffle (역방향 재생성) | Undo (상태 복원) | Add Lane |
| :--- | :--- | :--- | :--- |
| **Locked Lane** | 잠금 상태 유지, 재배치 대상에서 제외 | 잠금/해제 상태 포함 복원 | 새 레인은 일반 레인 |
| **Blind Lane** | 재배치 후 하단 칩 전체 다시 가려짐 (정보 리셋) | 원위치 복원 + 이동 전 상태로 다시 가려짐 | 새 레인은 일반 레인 (Blind 아님) |
| **Relay Node** | 순서 제약 유지, 대기 상태 초기화 후 재배치 | 대기 상태 포함 전체 복원 | 영향 없음 |
| **Overload Chip** | 같은 타입 Top Chip 레인이 존재하도록 배치 보장 (Solver 조건 적용) | 위치 복원 (Overload 속성 유지) | 새 빈 레인에 Overload Chip 단독 배치 불가 (규칙 유지) |

---

## 6. 기믹 복합 (Challenge 모드, Lv.101+)

Ch.6 이후 이전 챕터 기믹이 조합 등장합니다. 주요 복합 패턴:

| 복합 | 난이도 기여 | 비고 |
| :--- | :--- | :--- |
| Locked Lane + Relay Node | Locked Lane 해제 조건이 Relay Node 순서 선행 조건과 연계 가능 | 순서 설계 신중히 |
| Blind Lane + Relay Node | 어떤 타입이 완성될지 모른 채 순서를 맞춰야 함 | 최고난도 조합 |
| Overload Chip + Blind Lane | Overload Chip이 가려진 채 등장, Top Chip 전환 시 공개 | 정보 비대칭 극대화 |
| Locked Lane + Blind Lane | 해제 조건 타입이 어디 있는지 모름 | 중~고난도 |
