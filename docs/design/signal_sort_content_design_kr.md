# Signal Sort 콘텐츠 기획서 (Signal Sort Content Design)

본 문서는 `project-fill`의 핵심 게임 플레이인 **Signal Sort (시그널 소트)**에 대한 콘텐츠 및 밸런스 기획서입니다. 난이도 파라미터 설계, 스테이지 진행 구조, 챕터 마일스톤 보상 밸런스, 기믹 도입 스케줄을 다룹니다.
기믹 상세 사양은 [`signal_sort_gimmick_design_kr.md`](signal_sort_gimmick_design_kr.md)를 참조합니다.

---

## 1. 개요 및 레벨 디자인 원칙

- **목표**: 플레이어가 칩의 색상을 맞추는 직관적인 플레이 속에서 점진적인 도전 의식을 갖도록 난이도 곡선을 제어합니다.
- **스트레스 제어**: 제한 시간, 강제 실패 요소(이동 횟수 제한, 별 조건), 스태미나를 두지 않습니다. 오직 논리적인 정렬 논리와 기믹 조합만으로 난이도를 설계하여 릴랙싱 퍼즐의 정체성을 유지합니다.
- **다양성 확보**: Signal Type의 수, 레인 수, 초기 배치 꼬임 정도, 챕터별 신규 기믹을 통해 입체적인 레벨을 생성합니다.

---

## 2. 난이도 파라미터 설계

| 챕터 | 스테이지 | Signal Type 수 | Slot Lane 수 | 빈 레인 수 | 신규 기믹 |
| :---: | :---: | :---: | :---: | :---: | :--- |
| **Chapter 1** | Lv. 1–20 | 3–4개 | Type수 + 2 (5–6개) | 2개 | 없음 (기본 룰) |
| **Chapter 2** | Lv. 21–40 | 4–6개 | Type수 + 2 (6–8개) | 1–2개 | Locked Lane |
| **Chapter 3** | Lv. 41–60 | 6–7개 | Type수 + 1–2 (7–9개) | 1–2개 | Blind Lane |
| **Chapter 4** | Lv. 61–80 | 7–9개 | Type수 + 1 (8–10개) | 1개 | Relay Node |
| **Chapter 5** | Lv. 81–100 | 9–10개 | Type수 + 1 (10–11개) | 1개 | Overload Chip |
| **Challenge** | Lv. 101+ | 10개+ | Type수 + 1 | 1개 | 기믹 복합 조합 |

- **Signal Type 수**: 많아질수록 화면에 인지해야 하는 색상의 수가 늘어나 인지 부하가 커집니다.
- **빈 레인의 수**: 2개→1개로 줄어들 때 난이도가 비약적으로 증가합니다.
- **기믹 복합**: Ch.5 이후부터 이전 챕터 기믹이 조합 등장 가능합니다.

## 3. 챕터 Progression 및 마일스톤 보상

### 3.1. 챕터 밸런스 테이블

| 챕터 | 스테이지 범위 | 테마 비주얼 | 마일스톤 조건 | 골드 보상 | 부스터 보상 |
| :---: | :---: | :--- | :--- | :---: | :--- |
| **Chapter 1** | Lv. 1–20 | 네온 반도체 (Neo-Semi) | 챕터 내 모든 스테이지 최초 클리어 | 500골드 | Shuffle ×3 |
| **Chapter 2** | Lv. 21–40 | 글리프 노드 (Glyph-Node) | 챕터 내 모든 스테이지 최초 클리어 | 800골드 | Shuffle ×5, Add Lane ×1 |
| **Chapter 3** | Lv. 41–60 | 다이오드 패널 (Diode-Panel) | 챕터 내 모든 스테이지 최초 클리어 | 1,200골드 | Shuffle ×5, Add Lane ×2 |
| **Chapter 4** | Lv. 61–80 | 양자 코어 (Quantum-Core) | 챕터 내 모든 스테이지 최초 클리어 | 1,600골드 | Shuffle ×8, Add Lane ×3 |
| **Chapter 5** | Lv. 81–100 | 오버로드 넥서스 (Overload-Nexus) | 챕터 내 모든 스테이지 최초 클리어 | 2,000골드 | Shuffle ×10, Add Lane ×5 |

### 3.2. 마일스톤 보상 규칙
- **조건**: 챕터 내 모든 스테이지 최초 클리어. 이동 횟수 및 부스터 사용 여부 무관.
- **서버 검증**: `user_stage_progress` 테이블에서 해당 챕터의 모든 `stage_clear = true` 여부를 확인 후 보상 지급.

## 4. 스테이지 클리어 보상 (골드)

| 챕터 | 스테이지 범위 | 최초 클리어 보상 | 재클리어 보상 |
| :---: | :---: | :---: | :---: |
| Ch.1 | Lv. 1–20 | 30골드/스테이지 | 없음 |
| Ch.2 | Lv. 21–40 | 35골드/스테이지 | 없음 |
| Ch.3 | Lv. 41–60 | 40골드/스테이지 | 없음 |
| Ch.4 | Lv. 61–80 | 45골드/스테이지 | 없음 |
| Ch.5 | Lv. 81–100 | 50골드/스테이지 | 없음 |

**100스테이지 완주 기준 총 획득 추산**:
- 스테이지 클리어 합계: `(30+35+40+45+50) × 20 = 4,000골드`
- 챕터 마일스톤 합계: `500+800+1,200+1,600+2,000 = 6,100골드`
- **총 획득: 약 10,100골드**

**평균 플레이어 골드 소모 추산** (Shuffle 20회 + Add Lane 골드 구매 2회):
- `20 × 100 + 2 × 500 = 3,000골드`
- **순 잔액: 약 7,100골드** (Ch.6+ 챌린지 콘텐츠 싱크 예약)

## 5. 기믹 도입 스케줄 및 튜토리얼

| 챕터 | 신규 기믹 | 최초 등장 스테이지 | 튜토리얼 트리거 |
| :---: | :--- | :---: | :--- |
| Ch.1 | (기본 룰) | Lv.1 | 최초 진입 시 온보딩 튜토리얼 |
| Ch.2 | Locked Lane | Lv.21 | Lv.21 최초 진입 시 자동 재생 |
| Ch.3 | Blind Lane | Lv.41 | Lv.41 최초 진입 시 자동 재생 |
| Ch.4 | Relay Node | Lv.61 | Lv.61 최초 진입 시 자동 재생 |
| Ch.5 | Overload Chip | Lv.81 | Lv.81 최초 진입 시 자동 재생 |

- 튜토리얼은 해당 챕터 첫 스테이지 진입 시 1회만 자동 재생됩니다.
- 기믹별 상세 규칙은 [`signal_sort_gimmick_design_kr.md`](signal_sort_gimmick_design_kr.md)를 참조합니다.
