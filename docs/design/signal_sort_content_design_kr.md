# Signal Sort 콘텐츠 기획서 (Signal Sort Content Design)

본 문서는 `project-fill`의 핵심 게임 플레이인 **Signal Sort (시그널 소트)**에 대한 콘텐츠 및 밸런스 기획서입니다. 난이도 파라미터 설계, 스테이지 진행 구조, 챕터 마일스톤 보상 밸런스, 후반부/챌린지에 대비한 특수 기믹 확장안을 다룹니다.

---

## 1. 개요 및 레벨 디자인 원칙

- **목표**: 플레이어가 칩의 색상을 맞추는 직관적인 플레이 속에서 점진적인 도전 의식을 갖도록 난이도 곡선을 제어합니다.
- **스트레스 제어**: 제한 시간이나 강제 실패 요소(이동 횟수 제한 등)를 두지 않고, 오직 논리적인 정렬 논리만으로 난이도를 설계하여 릴랙싱 퍼즐의 정체성을 유지합니다.
- **다양성 확보**: 등장하는 Signal Type의 수, 사용 가능한 레인의 수, 초기 배치 상태의 꼬임 정도를 통해 입체적인 레벨을 생성합니다.

---

## 2. 난이도 파라미터 설계

스테이지 난이도는 기본적으로 아래 3가지 파라미터를 조절하여 결정됩니다.

| 난이도 구분 | Signal Type 수 (컬러 수) | Slot Lane 수 (총 레인 수) | 빈 레인 수 (Empty Lanes) |
| :--- | :--- | :--- | :--- |
| **초반 (Lv. 1 ~ 20)** | 3 ~ 4개 | Type 수 + 2개 (5 ~ 6개) | 2개 |
| **중반 (Lv. 21 ~ 60)** | 5 ~ 7개 | Type 수 + 1~2개 (6 ~ 9개) | 1 ~ 2개 |
| **후반 (Lv. 61 ~ 100)** | 8 ~ 10개 | Type 수 + 1개 (9 ~ 11개) | 1개 |
| **고난도 / 이벤트** | 10개 이상 | Type 수 + 1개 | 1개 (기믹 포함) |

- **Signal Type 수**: 많아질수록 화면에 인지해야 하는 색상의 수가 늘어나 검색 부하가 커집니다.
- **빈 레인의 수**: 플레이어가 칩을 임시로 임시 보관해 둘 수 있는 여유 공간으로, 빈 레인이 2개에서 1개로 줄어들 때 난이도가 비약적으로 증가합니다.

---

## 3. 챕터 progression 및 마일스톤 보상

스테이지는 챕터 단위로 묶여 관리되며, 플레이어에게 단기/장기 목표를 제공합니다.

### 3.1. 챕터 밸런스 테이블 (예시)
| 챕터 | 스테이지 범위 | 테마 비주얼 | 보상 상자 조건 | 상자 보상 구성 |
| :---: | :---: | :--- | :--- | :--- |
| **Chapter 1** | Lv. 1 ~ 20 | 네온 반도체 (Neo-Semi) | 챕터 내 모든 스테이지 별 3개 달성 | 500 골드, Undo 3개, Shuffle 1개 |
| **Chapter 2** | Lv. 21 ~ 50 | 글리프 노드 (Glyph-Node) | 챕터 내 모든 스테이지 별 3개 달성 | 1,000 골드, Undo 5개, Shuffle 3개, Hint 1개 |
| **Chapter 3** | Lv. 51 ~ 80 | 다이오드 패널 (Diode-Panel) | 챕터 내 모든 스테이지 별 3개 달성 | 1,500 골드, Undo 5개, Shuffle 5개, Hint 3개, Add Lane 1개 |
| **Chapter 4** | Lv. 81 ~ 100 | 양자 코어 (Quantum-Core) | 챕터 내 모든 스테이지 별 3개 달성 | 2,500 골드, Undo 10개, Shuffle 8개, Hint 5개, Add Lane 3개 |

### 3.2. 챕터 마일스톤 보상 상자 규칙
- **별 3개 조건**: 플레이어가 특정 스테이지를 Stuck 상태에서 구제받지 않고(또는 특정 이동 횟수(Moves) 이하로) 클리어할 때 별 3개를 획득합니다.
- **스태미나 제외**: 기존에 존재했던 스태미나 충전 및 "무제한 스태미나" 상품은 모두 삭제되었으므로, 마일스톤 보상은 **골드(소프트 커런시)**와 **부스터 아이템 4종 패키지**로만 순수 구성됩니다.
- **서버 검증**: 클라이언트가 상자를 오픈할 때, 서버는 `user_stage_progress` 테이블을 조회하여 해당 챕터의 모든 스테이지에서 실제 별 3개 획득 여부를 검증하고 보상을 지급합니다.

## 4. 특수 기믹 확장안 (Lv. 40 이후 순차 도입)

기본 룰의 단조로움을 극복하고 후반 레벨의 퍼즐 깊이를 더하기 위해 도입하는 특수 장치들입니다.

### 1) Locked Lane (잠긴 레인)
- **개념**: 레인 상단에 자물쇠 스킨이 씌워져 있어 최초에는 칩을 배치할 수 없는 레인입니다.
- **잠금 해제 조건**: 특정 지정된 Signal Type의 Complete Set이 먼저 완성되면 잠금이 풀리면서 일반 빈 레인으로 사용할 수 있게 됩니다.
- **도입 레벨**: Lv. 40 이후 권장

### 2) Frozen Chip (얼어붙은 칩)
- **개념**: 레인의 특정 위치에 배치되어 있어, 바로 위층의 칩이 이동하거나 보드 내에서 1회 이동이 일어나기 전까지는 선택하거나 이동할 수 없는 얼어붙은 칩입니다.
- **도입 레벨**: Lv. 60 이후 권장

### 3) Relay Node (릴레이 노드)
- **개념**: 칩을 정렬하는 순서에 제약이 걸린 기믹입니다.
- **규칙**: 반드시 상단 Signal Panel에서 예고된 A 타입이 먼저 Complete Set이 되어야 B 타입의 칩이 상단 패널로 흡수될 수 있습니다. 순서가 어긋난 세트 완성은 대기 상태에 빠집니다.
- **도입 레벨**: Lv. 80 혹은 챌린지 모드 권장

### 4) Overload Chip (과부하 칩)
- **개념**: 극성이 너무 강해 다른 빈 레인에는 단독으로 내려놓을 수 없고, 오직 같은 타입의 칩 위로만 올라갈 수 있는 제한이 걸린 칩입니다.
- **도입 레벨**: 마스터/챌린지 모드 권장
